# Escape-Log 프로젝트 메모리

## 작업 시작 전 필수 고지 (사용자 요구사항)
모든 작업 시작 전에 반드시 다음을 먼저 말해야 한다:
1. 현재 Phase 번호와 이름
2. 이번에 할 작업 내용 (무엇을, 왜)
3. 브랜치명 제안

## 현재 진행 상태
- 현재 Phase: **Phase 5 — UX 완성** 진행 중
- 작업 우선순위:
  1. 폼 유효성 검증 (ReviewCreateForm.vue)
  2. 로딩/에러/빈 상태 UI
  3. 사진 업로드 (Supabase Storage)
  4. v1 마감 (환경 변수, 배포, README)
